# Entrepreneurship

本分類涵蓋 Minimalist Entrepreneur（極簡創業家）方法論的完整工作流，包括：社群識別、商業構想驗證、MVP 建構、手動流程化、早期客戶獲取、定價策略、內容行銷計畫、可持續成長決策、公司文化與價值觀建立、以及業務決策審查。

這些技能均基於 Sahil Lavingia（Gumroad 創辦人）所著《The Minimalist Entrepreneur》一書的理念，適用於 bootstrapped 創業者、獨立開發者、或任何希望以小規模、可獲利方式建立事業的人。

## 適用場景

- 尋找商業方向、想了解「從哪裡開始」的創業者
- 有想法但不確定是否值得執行的人
- 準備建立 MVP 或被功能範圍卡住的開發者
- 有產品但需要找到第一批客戶的創辦人
- 需要設定或調整定價策略的人
- 已達初步 product-market fit、需要行銷計畫的人
- 面臨擴張、招聘或融資決策的創辦人
- 準備建立公司文化或招聘的團隊領導者
- 任何需要用極簡創業原則審視商業決策的人

## 技能清單

| 技能 | 說明 |
|------|------|
| `find-community` | 識別並評估可建立業務的目標社群 |
| `validate-idea` | 用極簡創業框架驗證商業構想 |
| `mvp` | 以最小投入建構 MVP（手動 → 流程化 → 產品化） |
| `processize` | 將產品構想轉化為可立即執行的手動流程 |
| `first-customers` | 制定策略找到前 100 個客戶 |
| `pricing` | 建立合理的定價模型與定價策略 |
| `marketing-plan` | 以內容建立受眾的極簡行銷計畫 |
| `grow-sustainably` | 從可獲利角度評估成長與擴張決策 |
| `company-values` | 定義公司價值觀與文化 |
| `minimalist-review` | 用極簡創業原則審查任何業務決策 |
