# Awesome Anti AI Slop

Use this skill as a curated reference for finding and evaluating anti-slop tools, skills, prompts, CLIs, apps, GitHub Actions, linters, and workflow rules.

## Workflow

1. Read `README.md` for the English catalog and `README_CN.md` when Chinese context is needed.
2. Start with the Best Picks and Starter Stacks sections for recommendations.
3. Match the user's domain: writing, Chinese writing, academic writing, frontend/UI/design, code/PR quality, apps/CLIs, or meta lists.
4. Apply the evaluation checklist before recommending a tool: avoid detector-bypass promises, synonym-only rewriting, fact drift, and marketing shells.
5. If the user asks to install one of the listed tools, inspect that linked project separately before adding it to this router.
