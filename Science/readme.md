# Science

此分類涵蓋 134 個科學與研究 skill（來源：[K-Dense-AI/scientific-agent-skills](https://github.com/K-Dense-AI/scientific-agent-skills)），橫跨以下領域：

- **生物資訊與基因體學**：序列分析（BioPython、pysam）、單細胞分析（Scanpy、scVelo）、基因調控網路（Arboreto）、變異註解（Ensembl VEP、ClinVar）、系統發育分析
- **化學資訊與藥物發現**：分子性質預測（RDKit、DeepChem）、虛擬篩選（DiffDock）、ADMET 分析、先導化合物優化（datamol、Molfeat）
- **蛋白質體學與質譜**：LC-MS/MS 處理（matchms、pyOpenMS）
- **臨床研究與精準醫學**：臨床試驗（ClinicalTrials.gov）、藥物基因體學（ClinPGx）、變異詮釋、藥物安全、臨床決策支援
- **醫學影像與數位病理**：DICOM 處理（pydicom）、全切片影像（histolab、PathML）
- **神經科學與電生理**：神經記錄分析（Neuropixels）
- **機器學習與 AI**：深度學習（PyTorch Lightning、Transformers）、強化學習（Stable Baselines3）、時間序列（aeon、TimesFM）、貝氏方法（PyMC）
- **材料科學、化學與物理**：晶體結構（Pymatgen）、代謝建模（COBRApy）、天文學（Astropy）、量子運算（Cirq、PennyLane、Qiskit、QuTiP）
- **工程與模擬**：CFD（FluidSim）、離散事件模擬（SimPy）、符號數學（SymPy）、MATLAB/Octave
- **資料分析與視覺化**：統計分析、網路分析（NetworkX）、地理空間（GeoPandas、GeoMaster）、大規模資料處理（Dask、Polars、Vaex）
- **實驗室自動化**：液體處理（PyLabRobot）、Opentrons、Benchling、LabArchives
- **多體學與系統生物學**：路徑分析（KEGG、Reactome、STRING）、PrimeKG、LaminDB
- **蛋白質工程與設計**：蛋白質語言模型（ESM）、醣工程
- **科學溝通**：文獻回顧、同儕審查、科學寫作、海報、投影片、示意圖、引用管理
- **科學資料庫**：統一查詢 78+ 公開資料庫（PubChem、ChEMBL、UniProt、PDB、AlphaFold 等）、DepMap、PrimeKG
- **基礎設施與平台**：Modal 雲端計算、GPU 最佳化、DNAnexus、LatchBio、OMERO
- **研究方法論與規劃**：假說生成、科學腦力激盪、批判性思考、研究補助撰寫
- **法規與標準**：ISO 13485 認證

適用於任何科學研究、生物醫學分析、藥物開發、臨床數據分析、實驗室自動化、科學寫作與溝通的場景。
