# references/wuxing-colors.md · 五行配色系统

## Wu Xing — 五行主色

每一行对应一种设计语言和品牌原型。选对行，配色就有了方向。

| 行 | 主色 Hex | OKLCH | 设计语言 | 适用品牌 |
|----|---------|-------|---------|---------|
| 木 Wood | `#4CAF65` | `oklch(64% 0.15 145)` | 有机感、曲线、留白 | 健康、教育、可持续 |
| 火 Fire | `#E8401A` | `oklch(55% 0.21 30)` | 高对比、大字、冲击感 | 科技创业、餐饮、运动 |
| 土 Earth | `#C9912A` | `oklch(65% 0.13 68)` | 衬线字体、对称、金感 | 金融、奢品、文化 |
| 金 Metal | `#B8C8D8` | `oklch(82% 0.03 220)` | 极简、网格、无装饰 | SaaS、医疗、汽车 |
| 水 Water | `#2B7CB0` | `oklch(52% 0.13 240)` | 深邃、流动、信任 | AI、金融科技、咨询 |

## 相生配色规则

选主色所在行，取其「生」的下一行做辅色，天然和谐：

```
木→火 / 火→土 / 土→金 / 金→水 / 水→木
```

## 品牌选行速查

- 能量感 / 创业 / 突破 → **火 Fire**
- 高端 / 传承 / 金融 → **土 Earth**
- 自然 / 健康 / 可持续 → **木 Wood**
- 精准 / 企业 / 极简 → **金 Metal**
- 智识 / 数据 / AI → **水 Water**
- 玄学感 / 最百搭 → **火+土（炎土并立）**
