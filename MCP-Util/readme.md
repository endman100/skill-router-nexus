# MCP Util

此分類涵蓋 MCP（Model Context Protocol）伺服器建置指南、工具列舉、設定、認證、呼叫、schema 管理、以及 Claude Plugin 整合。適用於 Agent 需要建置 MCP 伺服器、直接操作 MCP 基礎設施、檢查 schema、管理連線，或整合 Claude Code Plugin 的場景。

## 包含 Skills
- `mcp-builder` — MCP 伺服器建置最佳實踐指南（跨分類複製自 Coding）
- `mcporter` — mcporter CLI 工具，用於列舉、設定、認證、呼叫 MCP servers/tools

## 包含非 skill 資產
- `.claude-plugin/marketplace.json` — Claude Skill Marketplace 目錄清單
- `connect-apps-plugin/` — Claude Code Plugin：透過 Composio 連接 500+ 外部 App
